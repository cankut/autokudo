Options

Specifies how many times we should go back in time while searching activity feed. Strava limits 20 entry per depth. So If you choose "5", it will get you the most recent 100 (5x20) entries in your feed.
Specifies the feed polling interval duration. If you choose 10min, it will check your feed every 10 minutes automatically.
When using auto mode, If you don't want to see "Kudo All" button on Strava Dashboard header, check this box.
